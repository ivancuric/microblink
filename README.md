# Microblink JS dev task

## How to build

Install using

```sh
npm install
```

To run the app in `dev` mode with live-reloading run

```sh
npm run dev
```

To build a production bundle in `/dist` use

```sh
npm run build
```

Built files can be served locally by running

```sh
npm run serve
```
